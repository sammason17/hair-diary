
# Hair Diary (Next.js + MongoDB)

A simplified version of the "Hair Diary" appointment planner built with Next.js, TypeScript, MongoDB, and NextAuth.

- 15-minute slots from 07:00 to 20:00
- Columns for Stewart, Sue, and Notes
- Store appointments in MongoDB
- Email/password login via NextAuth Credentials provider
- Deployable to Vercel with minimal configuration
